# Easy-Ledger 极简记账
一款**纯离线**的个人资产记账应用，专为需要定期汇总、统计分散资产的用户设计，支持银行卡、网络支付（微信/支付宝等）、信用卡等各类资产的月度记录与趋势追踪，轻松掌握个人资产变化。

项目在线体验：https://xpfrank.github.io/Easy-Ledger/

## ✨ 功能特点
- 📴 完全离线使用，无需网络，数据不泄露
- 👛 灵活的账户管理，支持账户的添加、编辑与删除
- 📊 月度/年度双维度记账，资产数据分层统计
- 📈 可视化资产趋势图表，资产变化直观可见
- 📝 记账操作自动记录，所有修改可追溯、可查看
- 🎨 多主题支持，适配不同视觉使用偏好

## 📲 安装方法
### 方法 1：下载 APK 安装（推荐）
1. 进入项目仓库右侧「Releases」板块
2. 下载最新版本的 APK 安装包
3. 将安装包传输至安卓手机，点击即可完成安装

### 方法 2：本地自行打包
请查看项目内的「打包说明.md」文件，查看详细的打包步骤与操作指引。

## 📋 快速使用
1. 首次打开应用，先在**账户管理**中添加个人资产账户（银行卡、支付宝、微信、信用卡等）
2. 每月在**记账页面**录入各账户最新余额，支持快速复制上月数据
3. 在**趋势页面**查看资产月度/年度变化趋势，在记账记录中追溯操作历史

## 🔒 数据安全
- 所有资产数据均**本地存储**在设备中，无云端上传，保障隐私安全
- 支持数据**导出/导入**备份，防止数据意外丢失
- 卸载应用会清空本地数据，卸载前请务必完成数据导出备份

## 🛠 技术栈
- 前端框架：React + TypeScript
- 样式方案：Tailwind CSS
- 跨端打包：Capacitor（安卓APP打包）
