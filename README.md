# Easy-Ledger 余额快照

一款**纯离线**的个人资产记账应用，专为需要定期汇总、统计分散资产的用户设计，支持银行卡、网络支付（微信/支付宝等）、信用卡等各类资产的月度记录与趋势追踪，轻松掌握个人资产变化。

项目在线体验：https://xpfrank.github.io/Easy-Ledger/

---

## ✨ 功能特点

- 📴 **完全离线使用**，无需网络，数据不泄露
- 👛 **灵活的账户管理**，支持账户的添加、编辑、删除、自定义排序与分组
- 🏷️ **账户分类体系**，支持快速分类引导，资产类账户归入现金/稳健/投资/保障四类，非资产类账户可标记跳过
- 🚁 **资产驾驶舱**，首页核心看板，综合展示资产配置结构、波动控制、归因记录与资产健康评分
- 🏦 **资产配置结构**，按人生阶段（成长期/稳定期/退休期等）设定参考区间，实时对比当前配置偏差
- 💱 **多币种支持**，支持人民币（CNY）、美元（USD）、欧元（EUR）、港币（HKD）、日元（JPY）等主流货币自由切换，手动汇率换算，净资产统一汇总
- 📊 **月度/年度双维度记账**，资产数据分层统计，支持快速复制上月数据
- 📈 **可视化资产趋势图表**，资产变化直观可见，支持月度/年度归因分析
- 🌊 **账户余额桑基图**，资产流向与分布一目了然，直观展示各分类资产占比与流转关系
- 🎯 **年度目标设定与达成追踪**，资产健康度实时评估
- 📝 **记账操作自动记录**，所有修改可追溯、可查看
- 🎨 **多主题支持**，适配不同视觉使用偏好
- 🔒 **余额隐私保护**，一键隐藏所有页面敏感金额数据
- 🔄 **数据导入/导出**，支持 JSON/CSV 格式备份与跨设备迁移

## 📋 快速使用

1. **首次打开应用**，先在**账户管理**中添加个人资产账户（银行卡、支付宝、微信、信用卡等），可为每个账户设置对应的货币类型
2. **完成快速分类**：点击首页「X个账户待分类」或设置页引导，将资产类账户归入现金/应急、稳健储蓄、投资增值、保险保障四类；信用卡、借贷等非资产类账户可点击「跳过」
3. **每月在记账页面**录入各账户最新余额，支持快速复制上月数据
4. **在首页资产驾驶舱**查看资产配置结构、波动控制、归因记录与健康评分，点击「查看健康分析」看详细解读
5. **在趋势页面**查看资产月度/年度变化趋势，点击图表节点可查看详情
6. **在账户管理页面点击【资产流向】**，通过桑基图直观查看各分类资产的分布与占比
7. **设定年度目标**，追踪资产达成进度，查看资产健康度评估
8. **为资产变动添加归因标签**，精准记录每笔变化的原因
9. **开启「余额隐藏」功能**，所有页面敏感金额自动脱敏，保护隐私安全
10. **定期导出数据备份**，防止数据意外丢失

## 🔒 数据安全

- 所有资产数据均**本地存储**在设备中，无云端上传，保障隐私安全
- 支持数据**导出/导入**备份（JSON/CSV 格式），防止数据意外丢失，支持跨设备迁移
- 卸载应用会清空本地数据，卸载前请务必完成数据导出备份

## 🛠 技术栈

- 前端框架：React + TypeScript
- 样式方案：Tailwind CSS
- 跨端打包：Capacitor（安卓 APP 打包）
- 图表可视化：SVG 自定义实现（支持智能刻度、单位适配）
- 本地存储：localStorage（数据离线持久化）
