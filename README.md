# 记账本 APP

一个完全离线的个人记账应用。

## 功能特点

- ✅ 完全离线使用，无需网络
- ✅ 账户管理（添加、编辑、删除）
- ✅ 月度/年度记账
- ✅ 资产趋势图表
- ✅ 记账记录追溯
- ✅ 多主题支持

## 安装方法

### 方法 1：下载 APK 安装（推荐）

1. 在右侧 "Releases" 中下载最新版本的 APK
2. 把 APK 传到安卓手机
3. 点击安装即可

### 方法 2：自己打包

查看 `打包说明.md` 文件了解详细步骤。

## 使用说明

1. 首次打开应用后，先添加你的账户（银行卡、支付宝、微信等）
2. 每月记录一次账户余额
3. 查看资产趋势和记账记录

## 数据安全

- 所有数据保存在手机本地
- 支持导出/导入备份
- 卸载应用会丢失数据，请先导出备份

## 技术栈

- React + TypeScript
- Tailwind CSS
- Capacitor（打包成安卓 APP）
