# 记账本 APP 打包教程（完全离线版）

## 方法一：使用 GitHub Actions 自动打包（推荐，无需本地环境）

### 步骤 1：创建 GitHub 账号
1. 打开 https://github.com/
2. 点击 "Sign up" 注册账号

### 步骤 2：创建代码仓库
1. 登录后点击右上角 "+" → "New repository"
2. 仓库名填：`my-ledger-app`
3. 选择 "Public"
4. 点击 "Create repository"

### 步骤 3：上传代码
1. 在仓库页面点击 "uploading an existing file"
2. 上传以下文件（从 /mnt/okcomputer/output/app 目录）：
   - package.json
   - vite.config.ts
   - tsconfig.json
   - tsconfig.app.json
   - tsconfig.node.json
   - index.html
   - components.json
   - src/ 文件夹（整个文件夹）
   - public/ 文件夹（整个文件夹）

### 步骤 4：创建工作流文件
1. 在仓库中创建文件夹 `.github/workflows/`
2. 在该文件夹中创建文件 `build-apk.yml`，内容如下：

```yaml
name: Build Android APK

on:
  push:
    branches: [ main, master ]
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    
    steps:
    - name: Checkout code
      uses: actions/checkout@v4
    
    - name: Set up Node.js
      uses: actions/setup-node@v4
      with:
        node-version: '20'
        cache: 'npm'
        
    - name: Install dependencies
      run: npm ci
      
    - name: Build web app
      run: npm run build
      
    - name: Set up JDK 21
      uses: actions/setup-java@v4
      with:
        java-version: '21'
        distribution: 'temurin'
        
    - name: Setup Android SDK
      uses: android-actions/setup-android@v3
      
    # 仅同步 Web 代码到 Android 项目（无需 init/add，因为 android 目录已存在）
    - name: Sync Capacitor with Android
      run: npx cap sync android
        
    - name: Fix Android SDK versions (optional)
      run: |
        # 修改 variables.gradle
        cat android/variables.gradle
        
        # 将 compileSdkVersion 改为 35
        sed -i 's/compileSdkVersion = .*/compileSdkVersion = 35/' android/variables.gradle
        sed -i 's/targetSdkVersion = .*/targetSdkVersion = 35/' android/variables.gradle
        
        echo "=== Modified variables.gradle ==="
        cat android/variables.gradle
        
        # 同时修改 app/build.gradle
        sed -i 's/compileSdkVersion .*/compileSdkVersion 35/' android/app/build.gradle
        sed -i 's/targetSdkVersion .*/targetSdkVersion 35/' android/app/build.gradle
        
        echo "=== Modified app/build.gradle ==="
        cat android/app/build.gradle
        
    - name: Build Debug APK
      run: |
        cd android
        chmod +x gradlew
        ./gradlew assembleDebug --no-daemon
        
    - name: Upload APK to Artifacts
      uses: actions/upload-artifact@v4
      with:
        name: 记账本-Debug-APK
        path: android/app/build/outputs/apk/debug/app-debug.apk
        
    - name: Create Release
      if: github.event_name == 'push' && (github.ref == 'refs/heads/main' || github.ref == 'refs/heads/master')
      uses: softprops/action-gh-release@v1
      with:
        tag_name: v${{ github.run_number }}
        name: 记账本 v${{ github.run_number }}
        body: |
          自动构建的测试版本
          
          ## 安装方法
          1. 下载 `app-debug.apk`
          2. 传到安卓手机安装
          3. 如果提示未知来源，请允许安装
        files: android/app/build/outputs/apk/debug/app-debug.apk
      env:
        GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

### 步骤 5：运行打包
1. 点击仓库上方的 "Actions" 标签
2. 点击左侧的 "Build Android APK"
3. 点击右侧的 "Run workflow"
4. 等待约 5-10 分钟
5. 完成后在 "Artifacts" 中下载 APK 文件

## 方法二：使用 Android Studio 本地打包

### 需要安装的软件：
1. Node.js：https://nodejs.org/ （下载 LTS 版本）
2. Android Studio：https://developer.android.com/studio
3. Java JDK 17：https://www.oracle.com/java/technologies/downloads/

### 打包步骤：

1. **下载代码**
   ```bash
   # 把 app 文件夹复制到你的电脑
   ```

2. **安装依赖**
   ```bash
   cd app
   npm install
   ```

3. **构建网页**
   ```bash
   npm run build
   ```

4. **安装 Capacitor**
   ```bash
   npm install @capacitor/core @capacitor/cli @capacitor/android
   ```

5. **初始化 Capacitor**
   ```bash
   npx cap init "记账本" "com.ledger.app" --web-dir dist
   ```

6. **添加安卓平台**
   ```bash
   npx cap add android
   ```

7. **打开 Android Studio**
   ```bash
   npx cap open android
   ```

8. **在 Android Studio 中**
   - 等待 Gradle 同步完成
   - 点击菜单栏 "Build" → "Build Bundle(s) / APK(s)" → "Build APK(s)"
   - 完成后在 `android/app/build/outputs/apk/debug/` 找到 APK 文件

## 方法三：找朋友帮忙打包

如果你不想折腾，可以：
1. 把代码文件发给懂技术的朋友
2. 让他按照方法二帮你打包
3. 把 APK 文件传回给你安装

---

## 安装 APK 到手机

1. 把 APK 文件传到手机（微信、QQ、数据线都可以）
2. 在手机上点击 APK 文件
3. 如果提示"未知来源"，点击"允许"
4. 点击"安装"
5. 完成！

---

## 注意事项

- 首次安装后，APP 内的数据会保存在手机本地
- 卸载 APP 会丢失所有数据，记得先导出备份
- 这个 APP 完全离线，不需要网络权限
